# <a name="nuget-error-nu5038"></a>Ошибка NuGet NU5038
<pre>The readme file 'readme.txt' has an invalid extension. It must end in .md.</pre>

### <a name="issue"></a>Проблема

Файл сведений должен быть файлом Markdown и должен заканчиваться на "*. md*".

### <a name="solution"></a>Решение

Используйте расширение файла Markdown для файла сведений.
