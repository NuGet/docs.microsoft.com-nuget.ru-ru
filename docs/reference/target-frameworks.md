# <a name="target-frameworks"></a><span data-ttu-id="047e8-103">Требуемые версии .NET Framework</span><span class="sxs-lookup"><span data-stu-id="047e8-103">Target frameworks</span></span>

<span data-ttu-id="047e8-104">NuGet использует ссылки на целевую платформу в различных местах, чтобы точно определить и изолировать зависимые от платформы компоненты пакета:</span><span class="sxs-lookup"><span data-stu-id="047e8-104">NuGet uses target framework references in a variety of places to specifically identify and isolate framework-dependent components of a package:</span></span>
