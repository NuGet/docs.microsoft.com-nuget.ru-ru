# <a name="target-frameworks"></a><span data-ttu-id="f2d9c-103">Требуемые версии .NET Framework</span><span class="sxs-lookup"><span data-stu-id="f2d9c-103">Target frameworks</span></span>

<span data-ttu-id="f2d9c-104">NuGet использует ссылки на целевую платформу в различных местах, чтобы точно определить и изолировать зависимые от платформы компоненты пакета:</span><span class="sxs-lookup"><span data-stu-id="f2d9c-104">NuGet uses target framework references in a variety of places to specifically identify and isolate framework-dependent components of a package:</span></span>

Можно указать в пакете на необходимость включить в проект отдельные пакеты в зависимости от целевой платформы проекта.</span><span class="sxs-lookup"><span data-stu-id="f2d9c-106">[.nuspec manifest](../reference/nuspec.md): A package can indicate distinct packages to be included in a project depending on the project's target framework.</span></span>
